e-Medicine

Mini Project

e-Medicine is a system that can be used in medical shops to manage the medicines and its medical details using SQL and java. It’s a website which allows medical shop owner (Admin) to add or remove the name and details of the medicines currently available in the shop and the customers will be able to check for the availability of medicines in registered medical stores via online.
