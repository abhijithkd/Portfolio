Recent Projects

AR Flashcards

Design Project

AR flashcards is an application to make learning fun with the technology of Augmented Reality. They are interactive, effective and more entertaining for the students to learn. The project I designed is scanning different character or word or images at clarity will provide it’s 3D model and pronounces the letter, name and some small information about that image which are useful for children.

e-Medicine

Mini Project

e-Medicine is a system that can be used in medical shops to manage the medicines and its medical details using SQL and java. It’s a website which allows medical shop owner (Admin) to add or remove the name and details of the medicines currently available in the shop and the customers will be able to check for the availability of medicines in registered medical stores via online.
